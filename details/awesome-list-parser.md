## Overview

Parses Awesome lists to structured JSON.

## Features

- Live metadata enrichment
- Category/entry extraction

## Use Cases

- List maintenance audits
- Pipeline discovery

## Comparisons to Static GitHub

| Feature | Parser | Static GitHub |
|---------|--------|---------------|
| Output | JSON enriched | Markdown |
| Enrichment | Stars/archived | None |
| Automation | Full | Manual |
