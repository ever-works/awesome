## Overview

Curated aggregator of 300+ launch directories, verified and updated weekly.

## Core Features

- 300+ Curated Directories pre-loaded with quality directories
- Advanced Filtering by DR, category, pricing, dofollow status
- Instant Search across names, descriptions, categories
- Multi-Select Checklist for PDF or CSV export
- Domain Ratings with weekly automated updates via Moz API
- Community Voting with IP-based 'helpful' system
- Giscus Comments powered by GitHub Discussions

## User Features

- Optional Authentication via Google & GitHub OAuth
- Favorites to save personal collections
- Submission Tracking for directories submitted to
- Newsletter Integration with Listmonk
- Public Stats dashboard
- Screenshot Generation for social sharing

## Technical Features

- Static SPA with Vue.js 3 + Vite
- GitHub Pages hosting
- Weekly automation via GitHub Actions
- Privacy-first analytics with Pirsch
- WCAG 2.1 AA compliant and responsive with Tailwind CSS

## Tech Stack

- Frontend: Vue.js 3 (Composition API), Vite, Tailwind CSS
- Backend: Supabase (PostgreSQL + Auth + Realtime)
- Hosting: GitHub Pages
- Newsletter: Listmonk
- Analytics: Pirsch
- CI/CD: GitHub Actions
- APIs: Moz API
