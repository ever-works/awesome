## Overview

Directory of Directories is a curated GitHub repository that collects and lists meta-directories across various niches such as web directories, AI tools lists, SaaS aggregators, and submission sites. It lists over 300 directories with URLs ideal for promotion and SEO.

## Features

- Curated collection of 300+ directories
- Categorized by type including AI, SaaS, and more
- Update frequency for each listed directory
- Focus on submission sites for visibility

## Use Cases

- Directory submission for SEO and promotion
- Discovering aggregators and lists for product visibility
- Alternative to awesome lists, emphasizing promotional directories over open-source

## Pricing

Free and open-source (hosted on GitHub).
