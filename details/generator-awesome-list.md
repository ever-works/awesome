## Overview

CLI tool that scaffolds awesome lists following best practices.

## Features

- Generates README.md with proper formatting and sections
- Creates .github/CONTRIBUTING.md and ISSUE_TEMPLATE
- Ensures compliance with awesome-lint standards
- Quick setup via `yo awesome-list`
- Customizable templates for different list types

## Use Cases

- Automate creation of new awesome lists
- Standardize maintenance across multiple lists
- Quickly bootstrap community-curated repositories
- Enforce quality and consistency in list building

## Comparisons

| Tool | Auto-generates structure | Lint compliance | CLI-based |
|------|---------------------------|-----------------|-----------|
| generator-awesome-list | Yes | Yes | Yes |
| Manual creation | No | Manual | No |
| awesome-lint | No | Yes | Yes |
