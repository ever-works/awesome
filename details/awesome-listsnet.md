## Overview

Awesome-Lists.net is a searchable directory aggregating thousands of GitHub awesome lists by topic, stars, and activity.

## Aggregation

Indexes thousands of lists with topic and metric-based grouping.

## Features

- Filtering by topic and other criteria
- Sorting by stars, forks, activity
- Daily updates

## Use Cases

- Quick discovery of curated resources on specific topics

## Comparisons

| Feature | Awesome-Lists.net | GitHub Search |
|---------|-------------------|---------------|
| Filters | Awesome-specific | General |
| Updates | Daily | Real-time but unindexed |
