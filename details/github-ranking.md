# Github Ranking

Dynamic ranking of GitHub repositories sorted by stars and forks, with separate leaderboards for overall and per programming language. Features top 10 lists with links to full top 100, including metrics like stars, forks, language, open issues, description, and last commit date. Automatically updated regularly.
