# jnv/lists

**Source:** [jnv/lists on GitHub](https://github.com/jnv/lists)

## Description
jnv/lists curates lists of directories from GitHub and web, discovering directory websites, repos, and curated lists across topics.

## Features
- Extensive curated lists (technical/non-technical)
- Sourced from GitHub/external sites
- Multi-level organization (lists of lists)
- 130+ contributors
- CSV export
- Includes awesome-* and themed collections

## Pricing
Free open-source.
