# taskwarrior.nvim

Neovim plugin to edit Taskwarrior tasks in a buffer, render tasks as Markdown checkboxes, bulk-edit with Vim motions, diff-and-apply on save. Inspired by oil.nvim. Part of Awesome Neovim Note Taking / Cursorline updates.
