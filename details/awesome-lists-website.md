## Overview

A web application enabling users to make and share awesome lists—collections of links to interesting resources on a given topic, similar to popular GitHub awesome lists.

## Technology

- Backend: Phoenix framework written in Elixir
- Frontend: Bootstrap 4.0

## Features

- Create and manage awesome lists
- Photo upload functionality
- Email sending capabilities

## Status

No longer actively maintained by the author.
