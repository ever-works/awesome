## Overview

A meta-collection of awesome lists focused on libraries, tools, frameworks, and software across programming languages and related areas. Contributions are welcome to add new lists or categories.

## Categories

- **Programming Languages**: AutoIt, C, C++, Clojure, Common Lisp, D, DotNet, Elixir, Erlang, Go, Haskell, Java, JavaScript, PHP, Python, R, Ruby, Rust, Scala.
- **Frameworks, Platforms**: Frontend (Frontdesk, Frontend Development, etc.), Node.js, React, Ruby on Rails, Mobile (Android, iOS), Backend (Phalcon).
- **Related and Useful**: Editors (Awesome SublimeText), Environments (Awesome Dev Env), IoT (Awesome IoT), Shell (Awesome Shell), SysAdmin (Awesome Sysadmin), Talks (Awesome Talks), Machine Learning (several lists), Awesomes (self-referential and others).

## Contribution

Feel free to add new lists or categories; names don't have to start with 'awesome-'.
