# insto

Interactive OSINT CLI / REPL with 35+ slash-commands for Instagram: profile + media + followers + dossier + geo-fingerprint, shared-followers intersection, superfan ranking, location search, URL metadata resolution, posting-cadence histogram, Maltego CSV export. Token-based (HikerAPI). Part of Awesome Osint updates.
