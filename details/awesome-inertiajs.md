## Overview

Curated directory of Inertia.js ecosystem resources including adapters, presets, packages, articles, and examples.

## Features

- Client-side adapters for Vue, React, Svelte
- Server-side adapters for Laravel, Rails, Symfony
- Starter kits, boilerplates, and presets
- Libraries, plugins, and extensions
- Articles, tutorials, and code examples
