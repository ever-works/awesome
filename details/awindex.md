## Overview

Generates searchable Awesome indexes.

## Features

- Real-time source ingestion
- Faceted search dashboard
- SQLite exports

## Use Cases

- Directory maintenance
- Queryable discovery

## Comparisons to Static GitHub

| Feature | awindex | Static GitHub |
|---------|---------|---------------|
| Search | Faceted | None |
| Updates | Cached refresh | Static |
| Export | DB/JSON | None |
